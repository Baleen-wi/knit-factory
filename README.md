# 针织厂进货货款管理系统

这是一个根据针织加工厂真实进货/对账流程重构的网页版系统原型。Excel 只作为演示数据来源，不作为系统边界。

当前版本是“本地单机网页版”：不需要安装依赖，直接打开即可使用；数据保存在浏览器里。后续要做多人协作、登录、云端数据库，可以在现有数据模型上继续升级。

## 核心功能

- 进货明细：记录日期、厂家、款号、颜色、单件价格、数量、批次、备注、对账状态
- 自动计算：单笔总价、款号总货款、厂家总货款、时间段总货款、所有货款
- 厂家对账：按厂家统计进货货款、已付货款、未付余额、未对账笔数
- 款号分析：按款号和颜色统计数量、总货款、均价、最低价、最高价
- 收款登记：记录收款日期、厂家、金额、收款方式和备注
- 打版管理：记录每批货前的打版情况（日期、厂家、款号、颜色、状态、费用），统计打版总数和状态分布
- 全局筛选：按日期、厂家、款号、颜色、备注筛选，所有统计实时变化
- 数据中心：导出完整 JSON 备份、导入 JSON/CSV/Excel(.xlsx)、清空或恢复演示数据
- 云备份：支持 GitHub / Gitee 自动备份，测试连接、一键备份、从云端恢复
- 打印对账单：在"厂家对账"页面一键打印，自动适配当前筛选范围
- 备份提醒：超过 7 天未导出备份，顶部自动提醒

## 推荐使用方式

1. 先在“数据中心”导出完整备份，养成备份习惯。
2. 在“进货明细”录入每天到货。
3. 每次和厂家核对后，把相关记录标为“已对账”。
4. 在“付款登记”录入付款。
5. 在“厂家对账”看每个厂家的未付余额。
