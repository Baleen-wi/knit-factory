导入、导出、重置

支持导入格式：JSON 完整备份、CSV 文本、Excel .xlsx（自动识别表头）。 导入 Excel 时会把每个工作表都扫描一遍，自动提取进货记录和收款记录。

云备份

GitHub / Gitee 自动备份

如何配置？
Gitee/GitHub：创建私有仓库 → 生成 Token（权限勾选 repo / projects）→ 填入上方。
坚果云：登录网页版 → 账号信息 → 安全选项 → 生成第三方应用密码 → 填入上方。
建议创建一个单独的私有仓库/文件夹专门存放备份文件。
